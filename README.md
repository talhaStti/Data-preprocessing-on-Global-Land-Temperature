# Global Land Temperature Analysis

This repository contains Python code to preprocess and analyze the Global Land Temperature dataset. 

## Contents
- `temperature_analysis.py`: Python script for data preprocessing and analysis.
- `global_land_temperature.csv`: Dataset file.

## Requirements
- Python 3.x
- Matplotlib

## Usage
1. Clone the repository: `git clone <repository_url>`
2. Navigate to the repository directory: `cd Global-Land-Temperature-Analysis`
3. Run the Python script: `python temperature_analysis.py`
